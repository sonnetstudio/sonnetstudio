GitHub Pages 업로드 방법

1. 이 압축파일을 풀어주세요.
2. GitHub 저장소에서 'uploading an existing file'을 클릭합니다.
3. 압축을 푼 폴더 안의 index.html과 images 폴더를 함께 드래그해서 업로드합니다.
4. 아래쪽 Commit changes를 누릅니다.
5. 저장소 Settings > Pages에서 다음처럼 설정합니다.
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
6. Save를 누른 뒤 1~5분 정도 기다리면 아래 주소로 접속할 수 있습니다.

https://sonnetstudio.github.io/sonnetstudio/
